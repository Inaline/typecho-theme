# Inaline Typecho Theme
-----------------------------

## 简介

功能强大、美观、简洁的 Typecho 主题

https://gitee.com/inaline/typecho-theme

- 主题目前只支持 MySQL 数据库，后面会考虑支持 SQLite、PostgreSQL
- 现在后台还是 Typecho 风格，以后可能会添加 Inaline 与 Typecho 风格的选择
- 默认使用 SSR (服务端渲染), 可选开启 RESTful API 服务, 可以让别的使用同款主题的用户获取到你的文章 (跨域需 token, 或者后端开启白名单)
- 支持 SPA (单页应用) 模式, 开启后默认打开 RESTful API ( 这个还早，暂时不打算开发 )zzzzzzzzzzz
